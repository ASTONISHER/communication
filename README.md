## IoT Labs 소개
IoT 디바이스는 잘 다루지만 Web 개발에 익숙하지 않은 개발자를 위해 만들었습니다.
자신의 IoT 아이디어를 쉽게 구현 할 수 있도록 도움을 주려는 것이 이 프로젝트의 목적 입니다.

### 구현 범위

- Device 통신
- Data 저장
- Data 분석 (Dashboard)

## 진행상황

- Device 통신 : `진행중`
- Data 저장 : `예정됨`
- Data 분석 (Dashboard) : `예정됨`

### 진행상황 단계 표시
1. `예정됨` : 개발 범위에 포함되어 있음
2. `진행중` : 현재 개발 진행 중
3. `사용가능` : 개발이 완료되어 사용 가능
4. `이슈` : 개발이 완료 되었으나 문제가 발생한 상태
